# OFM-Sonso

This module provides function to control of Sonsos speaker via the unoffical local API.

## Platforms

* ARDUINO_ARCH_ESP32

## Defines

| Define                        | Default  | Description                           |
|-------------------------------|----------|---------------------------------------|
| OPENKNX_SONOS_DEBUG           | undef    | Enable debug log for SONSO lib        |
